문제 1. MLflow 모델 서빙에서 사용되는 주요 컴포넌트는 무엇인가?

가. MLflow Tracking Server 

나. MLflow Projects 

다. MLflow Models

라. MLflow Model Registry


문제 2. MLflow 모델 서버는 어떤 종류의 모델을 지원하는가? 

가. scikit-learn 모델 

나. XGBoost 모델 

다. TensorFlow 모델 

라. 모두 지원한다.


문제 3. MLflow 모델 서버에서 모델을 배포할 때 어떤 방식을 사용할 수 있는가? 

가. 단일 노드에 배포 

나. Kubernetes 클러스터에 배포 

다. AWS SageMaker에 배포 

라. 모두 가능하다.
